# YTM Lyrics + UI (Beta)

この拡張機能は、YouTube Musicのインターフェースを刷新し、歌詞同期機能を追加するために開発されています。なお、現状はベータ版であり不具合が複数存在しますのでご了承ください。

---

## 主な機能

*   **同期歌詞の表示**
    LRCLIB APIを利用し、再生位置に正確に同期した歌詞を表示。クリックすることでその行の部分に曲を移動させることができます
*   **全画面表示UI**
    アルバムアートワークから抽出した色を背景に適用し、Apple Music風のモダンなフルスクリーンプレイヤーを表示。
*   **検索ロジック改良**
    `GET /api/get`と`GET /api/search`を組み合わせ、より確実に歌詞を取得。
*   **レスポンシブ対応**
    ウィンドウ幅に応じてレイアウトを自動調整。PCでは情報量の多い横並び表示、モバイル（狭い幅）ではスクロールに最適化した縦並び表示に切り替え。なお、モバイル表示における既知の不具合が存在するため、スマートフォン等での使用は非推奨です。
*   **歌詞のキャッシュ**
    歌詞のAPIからの取得には数秒の遅延が発生するため、一度取得した歌詞はローカルに保存し高速で表示。

> **既知の不具合**
> 現在はベータ版であり、以下の既知の問題があります。
> *   **歌詞の取得精度** 多くの楽曲において、歌詞が見つからない、または誤った歌詞が表示されることが確認されています。
> *   **モバイル表示の挙動:* ウィンドウ幅を狭くした際（モバイル表示）、特定の操作を行うとプレイヤーバーのボタンが消えたり、表示が崩れたりするバグが確認されています。

## インストール方法

現在、Chromeウェブストアでは公開されていません。以下の手順で手動インストールを行ってください。

1.  **ダウンロード**
    *   このリポジトリのページ上部にある緑色の **`<> Code`** ボタンをクリックします。
    *   **`Download ZIP`** を選択し、適当なフォルダに解凍します。
2.  **拡張機能の読み込み**
    *   ブラウザのアドレスバーに `chrome://extensions/` と入力します (Chrome/Edge/Brave等)。
    *   右上の **「デベロッパーモード」** スイッチをオンにします。
    *   **「パッケージ化されていない拡張機能を読み込む」** をクリックします。
    *   解凍したフォルダ（`manifest.json` が入っているフォルダ）を選択します。
3.  **完了**
    *   YouTube Musicを開き、音楽を再生すると自動的に新しいUIが適用されます。

## プロジェクトの背景について

このプロジェクトは、権利関係の整合性とコードベースの安定性を保証するために行った**ハードフォーク**です。

### フォーク元について
本ソフトウェアは [naikaku1/YouTube_Music-Moden-UI](https://github.com/naikaku1/YouTube_Music-Moden-UI) を元としています。

### ハードフォークの理由
元のリポジトリにおいて、CLAや権利関係が不明瞭なプルリクエストのマージによる将来的なリスクを抱えたコードの混入が懸念されました。
これを排除し、クリーンな状態から開発を継続するため、Naikaku氏のみが変更を行った最後のコミット **`5ff9249`** を起点として新たにこちらをしました。

フォーク後、デザインの変更などを含めた大幅なリファクタリングを実施しています。
